# SpeechShield

**SpeechShield** — умная защита вашего стрима от нецензурной лексики в реальном времени.

## О проекте

SpeechShield автоматически фильтрует ненормативную лексику во время стрима и защищает ваш канал от блокировок. Программа работает в реальном времени, анализируя аудиопоток и заглушая нежелательные слова.

## Возможности

- Фильтрация в реальном времени
- Статистика срабатываний
- Поддержка 24/7
- Пробный период 10 дней бесплатно — без привязки карты

## Технологии

- React 18
- TypeScript
- Vite
- CSS Modules

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```
